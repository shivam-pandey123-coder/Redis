# Redis

## About

This repository contains learning projects and implementations related to Redis, a powerful in-memory data structure store. The project explores various use cases and patterns including login systems with OTP (One-Time Password) functionality with TTL (Time-To-Live) management.

## Project Overview

The repository demonstrates practical applications of Redis including:
- **Session Management**: Implementing secure authentication systems
- **OTP with TTL**: Learning how to use Redis for time-bound one-time passwords
- **Data Structures**: Exploring Redis data types and operations

## Key Concepts

- **Redis**: In-memory data store for fast data access
- **OTP (One-Time Password)**: Secure authentication mechanism
- **TTL (Time-To-Live)**: Automatic expiration of keys in Redis
- **Session Management**: Managing user sessions efficiently
